# What is ZSH? Why switch? * Customizable * Compatible * Comfortable
# Quick Tour of some features
## Tab Completions
### Tab completion of arguments Tab completion works for command line arguments, not just commands.
### Tab completion in `cd` Tab completion works for `cd`, not just commands. You can do nice things like `cd /u/lo/b` and get into `/usr/local/bin`
### tab complete process IDs You can do things like `kill firefox -TAB`
### Tab complete `ssh` hostnames `ssh blah` -TAB
### Tab complete `git` branches
### Even more tab completions The zsh-users mailing list maintains a repo with _tons_ of completions at [https://github.com/zsh-users/zsh-completions](github.com/zsh-users/zsh-completions)
## Command History
### Command-specific history Type a command, then up-arrow through the history of commands starting with that
### Shared History `setopt share_history` Now all your shells on the machine share command history. No more closing a window and losing the history in it
## Improved globbing
### Recursive file globbing `ls ~/z/**/*.sh` The `**` looks in all the subdirs, not just one level
### Globbing can specifically target Files or Directories `ls **/*(.)` Just the files `ls **/*(/)` Just directories
### Globbing can target by size `ls -lah **/*(.Lg-1)` Files < 1 GB `ls -lah **/*(.Lm+10)` Files > 10 MB
## Miscellaneous cool bits
### Syntax highlighting Downloadable at [github.com/zsh-users/zsh-syntax-highlighting](https://github.com/zsh-users/zsh-syntax-highlighting)
### Autosuggestions Download at [github.com/zsh-users/zsh-syntax-highlighting](https://github.com/zsh-users/zsh-syntax-highlighting)
### AutoCD `setopt autocd` Just type the name of a directory to `cd` to it.
### Mass file renames `autoload -U zmv` to enable `zmv '(*).foo' '$1.txt'`
# Frameworks
## oh-my-zsh First framework, has version 1.0 issues [https://ohmyz.sh](https://ohmyz.sh) * Slow to start new sessions * Using 3rd party plugins is more painful
## zgen [https://github.com/tarjoilija/zgen](https://github.com/tarjoilija/zgen) * Much faster session starts * Can load oh-my-zsh plugins * Can automagically clone 3rd party plugins
## zsh-quickstart-kit I wrote a quickstart - [https://github.com/unixorn/zsh-quickstart-kit](https://github.com/unixorn/zsh-quickstart-kit) that includes all the stuff I've mentioned in the talk, and even more goodies.
